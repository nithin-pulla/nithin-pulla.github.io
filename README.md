# Portfolio Project

This project is a React-based portfolio website styled with Tailwind CSS and powered by Next.js.

## Prerequisites

- Node.js (v18 or higher)
- npm or yarn

## Getting Started

1.  **Install Dependencies**

    ```bash
    npm install
    ```

2.  **Run Development Server**

    ```bash
    npm run dev
    ```

    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

3.  **Build for Production**

    ```bash
    npm run build
    npm start
    ```

## Project Structure

- `Pages/`: Page components (migrated to `pages/` for Next.js routing).
- `Components/`: Reusable React components.
- `styles/`: Global styles and Tailwind configuration.
- `public/`: Static assets.

## Customization

- Update `Components/portfolio` files to change content.
- Update `tailwind.config.js` to modify the theme.
